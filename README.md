# ✨ Advanced Hand Tracking AR - Neon Aura

An interactive **Augmented Reality (AR) web experience** that uses real-time hand tracking to create stunning neon visual effects and gesture-based interactions.

## 🚀 Features

- 🖐️ Real-time hand tracking using MediaPipe  
- ✨ Neon visual effects & particle system  
- ⚡ Gesture detection (Pinch, Open Hand, Fist)  
- 🌈 Multiple themes (Rainbow, Cyberpunk, Lava, Ocean, Galaxy)  
- 🔊 Interactive audio feedback using Web Audio API  
- 🎨 Smooth animations using HTML Canvas  
- 📱 Fully responsive UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- MediaPipe Hands  
- Canvas API  
- Web Audio API  

---

## 🎮 How It Works

- Detects hand landmarks using MediaPipe  
- Tracks finger movements in real-time  
- Generates:
  - Particle effects  
  - Shockwaves on pinch gesture  
  - Neon trails & lightning effects  
- Uses distance between fingers to detect gestures
